💻 使用示例
# CLI 模式：一行安装 $ gitinstall ollama/ollama $ gitinstall comfyanonymous/ComfyUI --dir ~/AI $ gitinstall AUTOMATIC1111/stable-diffusion-webui --dry-run # 对话模式（OpenClaw Skill） 用户: 帮我装 ComfyUI AI: 检测到 macOS + Apple M3 + Python 3.11，生成 6 步安装计划... 用户: 好的，开始安装 AI: 正在执行... ✅ 安装完成！运行 python main.py 启动 # 高级功能 $ gitinstall audit owner/repo # 依赖安全审计 $ gitinstall doctor # 系统诊断 $ gitinstall updates check # 检查更新 $ gitinstall web --port 9090 # 启动 Web 界面 $ gitinstall chain owner/repo # 依赖链可视化
